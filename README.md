# Panda Skills Community

Public skill catalog for [ai-assitant-panda](https://github.com/jinqishen0725/ai-assitant-panda).

## Layout

- `index.json` — curated store index (V1)
- `skills/` — folder-backed skills scanned by the app (V2)
- `packages/` — optional `.panda-skill.zip` packages referenced by the index

## Install in Panda

Open **Skills → Community**, pick a skill, tap **Install**.
